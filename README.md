# VIT Course Registration

Small demo project for VIT course registration.

## Run locally

1. Install dependencies (if any):

```powershell
npm install
```

2. Start the server:

```powershell
node server.js
```

3. Open `index.html` in your browser or visit the server URL if `server.js` serves it.

## Notes

- Files to ignore are in `.gitignore`.
- To push to GitHub, create a repository and run `git remote add origin <URL>` then `git push -u origin main`.
